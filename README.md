#### 一些小技巧

将本组织仓库加入远程仓库配置便于 fetch
`git remote add upstream https://github.com/ZAFU-PCHospital/ZAFU-PCHospital-Doc.git`

不要使用 github 的 sync 功能，如果此仓库有更新请

```
git fetch upstream
git rebase upstream/main
git push origin --force-with-lease
```

以上三步只使用于没有冲突的情况，如果有冲突

```
git fetch upstream
git rebase upstream/main
# 遇到冲突后：
git add ...
git rebase --continue
git push origin --force-with-lease
```
