# 维修流程

## 简要通用维修流程

1. 询问客户要处理的业务
2. 确认业务，让客户填写表格。表格很重要！可以被看作是免责声明，当用户签字之后即生效
3. 维修结束之后，把表格的剩余部分填写完，视作一次维修纪录
4. 维持桌面整洁，业务结束后及时清理桌面，把工具放回原位

---

## 详细通用维修流程

### 流程总览

```
线上/线下接单  →  免责声明  →  诊断排查  →  维修操作  →  客户验收  →  记录归档
                                                            ↓（修不好）
                                                    老手兜底 / 转官方售后
```

### 接单

- **线上**：通过 QQ 群 / 微信群等渠道预约，说明故障现象。
- **线下**：固定值班时间在集贤 319 接待，直接带电脑来即可。

### 诊断排查

- **谁接谁负责到底**，接待者全程跟踪该机。
- 思路：先听描述 → 复现故障 → 从软件到硬件逐步排查 → 不确定时咨询老手，**不要盲目动手**。

### 维修操作

- 工具准备就绪再动手（PE 盘、螺丝刀套装、备件等）。
- 涉及数据风险的操作，需再次口头确认数据已备份。
- 原则：**自己有多少能力揽多少活，宁可少做不可做错。**

### 修不好的情况

遇到超出能力范围的问题，处理优先级：

1. **老手兜底** — 找组织内更有经验的成员接手
2. **转介官方售后** — 参考知识沉淀中的品牌售后链接和客服热线
3. 不要强行维修，更不要推荐没有把握的第三方

### 客户验收

- 维修完成后，**让客户当场操作测试**，确认故障已解决。
- 对所做操作给予简要说明，提醒后续注意事项（定期备份、不装来路不明的软件等）。

### 记录归档

- 修完后在在线表格/系统中记录：故障原因、处理步骤、最终结果。
- 维持桌面整洁，工具放回原位。

## 具体维修项目流程

### 清灰

1. 先检查机子有没有关机，如果没有关机就帮客户关机
2. 利用桌面上的（电动）螺丝刀取下电脑背面的螺丝（注：个别螺丝是防脱螺丝，很难取下来）
3. 取下螺丝之后，用吸盘或者撬棒将电脑盖子一点点撬开，当撬开的差不多时，取下电脑盖子（注：注意安全，个别机子盖子很难拆开（点名 omen），不要暴力拆开从而损坏了客户机子）
4. 用鼓风机对准风扇，先吸后吹，一边用鼓风机吹吸，一边用小刷子刷风扇，直至风扇上无明显灰尘
5. 关掉鼓风机，固定住风扇，用小刷子慢慢刷，将风扇上的灰尘清理干净
6. 扣上盖子，拧回螺丝，开启电脑检查（注：个别电脑在断过电或者开盖之后需要插上电源才能开机，需客户自带充电线）

### 换硅脂

1. 先检查机子有没有关机，如果没有关机就帮客户关机
2. 利用桌面上的（电动）螺丝刀取下电脑背面的螺丝
3. 取下螺丝之后，用吸盘或者撬棒将电脑盖子一点点撬开，当撬开的差不多时，取下电脑盖子
4. 找到电池排线，断开电池排线，左右摇摆着将电源线断开
    > [!Note]
    > 天选系列有致命小铁片，如果对手法没有自信可以断开电池端的排线而不是主板端的排线
5. 用螺丝刀拆除散热模块上的螺丝，按照螺丝旁边的数字顺序拧螺丝，先把每个螺丝都拧松一点，然后再将螺丝一个一个拧下来，接着拆下散热模块
6. 用酒精片擦除散热模块以及CPU上的残留硅脂
7. 接下来介绍两种涂抹方法：
    1. 硅脂，将硅脂均匀涂到CPU上
    2. 相变片，先将相变片与CPU大小进行比对，裁剪至合适大小，先撕去相变片一侧的塑料薄膜，将相变片置于CPU之上，再撕下另一侧的塑料薄膜
8. 然后将散热模块盖上，把螺丝按照倒序一颗一颗拧回去，每颗螺丝都拧一点，最后再拧紧，重新接回电源
    > [!Note]
    > 散热模块的螺丝一定要拧紧，这关乎散热效果
9. 扣上盖子，拧回螺丝，开启电脑检查（注：个别电脑在断过电或者开盖之后需要插上电源才能开机，需客户自带充电线）
10. 扣上盖子，拧回螺丝，重启电脑（注：个别电脑需要插上电源才能重启，需客户自带充电线）

## 客户沟通话术

> **沟通铁三角：态度温和 · 立场坚定 · 不卑不亢。**

### 黄金原则

> **维修之前先管人，再管电脑。**
>
> 让客户觉得被尊重、被认真对待——哪怕问题没解决，他也不会怪你。

### 免责声明与数据风险

**场景**：客户要求清 C 盘 / 重装系统，但数据没备份。

| 要点     | 参考话术                                                                                                           |
| -------- | ------------------------------------------------------------------------------------------------------------------ |
| 开场确认 | "你电脑上有哪些比较重要的文件？比如桌面上、文档里、微信聊天记录这些？"                                             |
| 说明风险 | "重装系统 / 清理 C 盘会把桌面和文档里的文件全部清掉，**不可恢复**。论文、照片、作业这些如果有的话，一定要先备份。" |

### 客户质疑方案

| 要点       | 参考话术                                                                                               |
| ---------- | ------------------------------------------------------------------------------------------------------ |
| 解释原因   | "目前这个问题最可能的原因是系统文件损坏 / 垃圾太多 / 驱动冲突，重装是最干净彻底的方案。"               |
| 回应担忧   | "重装后原来的软件确实要重新装。如果你有哪几个特别重要找不到安装包的，我可以先帮你记下来，装完一起找。" |
| 让客户决定 | "如果你暂时不想重装，我也可以先试试别的办法，但效果不一定理想。你决定。"                               |

### 修不好时的拒绝与转介

| 要点     | 参考话术                                                                        |
| -------- | ------------------------------------------------------------------------------- |
| 诚实说明 | "这个情况比较复杂 / 涉及硬件维修，我们这边目前处理不了，强行弄可能更麻烦。"     |
| 给出方向 | "建议去**官方售后**，他们可以免费检测，保修期内还能免费维修。"                  |
| 提供信息 | "你电脑是 [品牌]，售后电话 [热线]，最近售后点在 [地点]。需要帮你查具体地址吗？" |
| 老手兜底 | "不急的话我问问更有经验的同学，看他能不能帮你看。"                              |

### 应对固执/傲慢的客户

| 原则     | 要点                                                                    |
| -------- | ----------------------------------------------------------------------- |
| 保持冷静 | 不急不躁，语速放慢。**绝不跟客户对呛。**                                |
| 坚守底线 | "我理解你的想法。但按我的经验，这样做可能导致 [风险]，我不能冒这个险。" |
| 给台阶   | "要不你回去考虑一下？我们每周都在，想好了随时来。"                      |
| 终极退路 | "如果你觉得不合适，完全没问题，可以去官方售后，他们更专业。"            |

> 绝对不要在客户在场时发表极端言论、贬低客户或背后议论被听到。
