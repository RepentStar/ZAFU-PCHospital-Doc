# 常用快捷键与技巧

掌握常用快捷键可以大幅提升日常操作效率，以下整理了 Windows 系统中最常用的快捷键与实用技巧

> [!Note]
> 文中的快捷键都不区分大小写

## 基础快捷键

| 快捷键           | 功能                                |
| ---------------- | ----------------------------------- |
| `Ctrl + C`       | 复制                                |
| `Ctrl + X`       | 剪切                                |
| `Ctrl + V`       | 粘贴                                |
| `Ctrl + Z`       | 撤销                                |
| `Ctrl + Y`       | 重做                                |
| `Ctrl + A`       | 全选                                |
| `Ctrl + S`       | 保存                                |
| `Ctrl + F`       | 在当前页面/文件内搜索               |
| `Ctrl + H`       | 在当前页面/文件内搜索               |
| `Alt + F4`       | 关闭当前窗口/程序                   |
| `F2`             | 重命名选中文件                      |
| `Delete`         | 删除文件（移至回收站）              |
| `Shift + Delete` | 永久删除（不经过回收站）**慎重!!!** |

## 任务管理器

当电脑出现卡死、程序无响应时，可以使用快捷键快速调出任务管理器强制结束程序。

按下 `Ctrl + Shift + Esc` 直接打开任务管理器，找到无响应的程序，右键选择结束任务即可。

也可以使用 `Ctrl + Alt + Delete` 进入安全选项页面，再点击任务管理器。

> [!Note]
> 使用 `Ctrl + Shift + Esc` 是最保险的，在系统主进程死机的时候 `Ctrl + Alt + Delete` 是无响应的，但是任务管理器会一直监听 `Ctrl + Shift + Esc`

# 常见问题

## 快捷键无效

请检查是否开启了输入法的快捷键占用，或游戏/软件的全局快捷键冲突，可尝试切换输入法后重试。

## 剪贴板历史无法使用

`Win + V` 首次使用需要手动开启，点击弹出窗口中的**开启**按钮即可启用剪贴板历史功能。
